# 方法论做诗

将复杂的思想、框架、方法论提炼为可背诵、可默念的诗句或口诀。

## 风格参考

参考 `./wei-qing-examples.md` 获取韦青的全部方法论做诗范例和风格特征。

## 工作流程

### 第一步：理解源材料

1. 通读用户提供的内容（文章、对话、框架、笔记等）
2. 提取核心概念和关键词，列出清单
3. 识别概念之间的关系：递进、并列、因果、时空交叉
4. 向用户确认：哪些是最核心的、必须保留的概念

### 第二步：选择口诀形式

根据概念数量和关系，选择合适的形式：

| 形式 | 适用场景 | 示例 |
|------|----------|------|
| **三字诀** | 3个并列或递进概念 | 认错→知错→改错 |
| **五字诀** | 5个概念的缩写或链条 | Think biG (SCBIG) |
| **七字句** | 7个递进概念，一字一义 | 知旨定境安律德 |
| **四句歪诗** | 多维度的完整框架 | 四句七言组合 |
| **递进句式** | 层层深入的逻辑 | 大动不如小动… |
| **否定递进** | 纠正常见误解 | 听到≠学到≠做到≠做好 |
| **三句纲领** | 操作层面的行动指引 | 讲一错、防二错、罚多错 |

向用户推荐形式并确认。

### 第三步：提炼与创作

核心原则：

1. **一字一义**：每个字（或词）必须对应一个明确的、可操作的概念
2. **有结构**：字与字之间有递进、并列或时空交叉关系，不是随意堆砌
3. **可默念**：读起来顺口，适合在散步、冥想时反复玩味
4. **有出处**：优先从经典（古文、名著、科学概念）中提炼用字
5. **可拆解**：每个字都能展开为一段完整的解释
6. **不空洞**：避免大而空的字眼（如"真善美""高大上"），要具体、可执行
7. **谦逊命名**：可以自称"顺口溜""歪诗""X字诀"，不自封为经典

创作步骤：

```
源材料 → 概念清单 → 筛选核心概念 → 寻找每概念的一字代表
→ 排列组合（试不同顺序） → 检查韵律和口感 → 添加结构维度标注
```

### 第四步：注释与交付

为每个口诀提供：

1. **口诀全文**：可直接背诵的版本
2. **逐字拆解**：每个字的含义、来源、操作含义
3. **结构说明**：字与字之间的关系（递进/并列/时空）
4. **使用场景**：什么时候默念、什么时候拿出来用
5. **源材料映射**：每个字对应源材料的哪个段落/概念

输出格式示例：

```markdown
## [口诀名称]（[X]字[形式]）

> [口诀全文]

### 逐字拆解
- **[字1]**：[含义]（源自：[出处]）
- **[字2]**：[含义]（源自：[出处]）
- ...

### 结构
[说明字与字之间的维度关系]

### 使用场景
[何时默念/使用]
```

## 质量检查清单

创作完成后，逐项检查：

- [ ] 每个字都有明确、不重叠的含义
- [ ] 字与字之间有逻辑关系，不是随机拼凑
- [ ] 读起来顺口，不拗口
- [ ] 能在10秒内背下来
- [ ] 每个字都能展开讲3分钟以上
- [ ] 没有空洞的口号字眼
- [ ] 用字优先来自经典或学科术语
- [ ] 有清晰的维度标注（空间/时间/递进/并列）

## 注意事项

- 如果源材料概念太多（>10个），先帮用户做减法，聚焦到最核心的3-7个
- 如果用户给的是英文内容，可以尝试中英双版本（如 SCBIG 的方式）
- 鼓励用户创作后实际默念几天，根据体感调整用字
- 一个主题可以有多个口诀（如三错法有核心三字 + 操作纲领两层）
- 不要一次给太多，一个主题1-3个口诀足够
