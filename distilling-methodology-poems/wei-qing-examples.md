# 韦青方法论做诗范例集

本文件收录微软中国CTO韦青的全部方法论做诗/口诀实例，作为风格参考。

## 一、四句歪诗（七字四句绝）——主框架

韦青每天散步前默念一遍，称之为"长读长新"：

```
知旨定境安律德    ← 学习的功夫（源自《大学》七证法）
想能应可已正将    ← 决策的功夫（韦青原创）
念念不忘初常本    ← 做事的底线（初心、常识、本分）
终身学问思辨行    ← 修炼的路径（源自《中庸》五步法）
```

### 逐句拆解

**第一句：知旨定境安律德**
- 浓缩《大学》的修身次第：格物致知→止于至善→知止而后有定→定而后能静→静而后能安→安而后能虑→虑而后能得
- 七个字 = 七个递进阶段
- 属于"学习"维度

**第二句：想能应可已正将**
- 空间/伦理维度（四方）：想(want) 能(can) 应(should) 可(permitted)
- 时间维度（三段）：已(past) 正(present) 将(future)
- "我站在中间，东西南北"——先从四个维度审视，再放到时间轴上判断
- 属于"决策"维度

**第三句：念念不忘初常本**
- 初 = 初心（original intention）
- 常 = 常识（common sense）
- 本 = 本分（duty）
- 做事的时候不追求伟大耀眼，只做本分，但本分源于初心
- 属于"行动"维度

**第四句：终身学问思辨行**
- 浓缩《中庸》五步：博学之→审问之→慎思之→明辨之→笃行之
- "终身"强调这是一辈子的修炼
- 属于"修炼"维度

## 二、三错法

**核心三字**：认错 → 知错 → 改错

- 认错：默认态度，我可能是错的（来自纳德拉 learn-it-all 文化）
- 知错：依靠机器帮助发现哪里错了（系统太复杂，人无法独自判断）
- 改错：由人指挥机器一起修正

**操作纲领**（三句口诀）：
```
讲一错    ← 奖励/表扬第一次犯的新错（鼓励探索）
防二错    ← 建立机制防止重复犯同样的错
罚多错    ← 多次重复的错误必须处罚
```

## 三、五信论

五个"信"字组成的认知链条：

```
信息 → 信仰 → 信心 → 信任 → 信誉
```

- 信息：宇宙由物质、能量、信息构成，信息是基础
- 信仰：在结果未证明前仍愿意行动
- 信心：在实践过程中逐步建立
- 信任：别人愿意把事交给你
- 信誉：源于知行合一的积累，是信任的基础

配套评判标准（引用古语）：
```
察其言、观其行、审其心
```

## 四、Think biG（SCBIG）

五种思维模式的英文缩写，凑成 "think big"：

```
S = System Thinking      系统思维（钱学森）
C = Critical Thinking    审辨思维（韦青刻意不译为"批判思维"）
b = Bayesian Thinking    贝叶斯/概率思维
i = Inversion Thinking   逆向思维（芒格）
G = Growth Mindset       成长型思维（Carol Dweck / 纳德拉）
```

## 五、站桩口诀

```
大动不如小动
小动不如微动
微动不如不动
不动之动
乃生生不息之动
```

韦青将武术修炼原则映射到人生哲学：粗动不如精动，精动不如静动，静中之动才是真正的生命力。

## 六、Peter Henn 智慧诗（韦青引用）

```
What is wisdom?
Error, error, and error
But less, less, and less
```

韦青用来支撑三错法的哲学基础：error（偏差）不是 mistake（错误），智慧是不断偏差但每次偏得更少。

## 七、无信论（原则性口诀）

核心主张：不相信被动学习能适应下一代挑战，必须知行合一。

```
听到不等于学到
学到不等于做到
做到不等于做好
知识不等于智慧
```

## 风格特征总结

| 特征 | 说明 |
|------|------|
| **字数** | 以七字为主（对应七言绝句），也有三字、五字 |
| **来源** | 从经典中提炼（大学、中庸）或自创 |
| **结构** | 每字对应一个概念，形成递进或并列关系 |
| **功能** | 用于日常默念、自我提醒、决策框架 |
| **命名** | 自称"顺口溜""歪诗""四句绝""X字诀" |
| **态度** | 谦称"歪诗"，不以大师自居 |
| **融合性** | 中西融合（古典文言 + 现代工程/科学概念） |
| **可操作** | 每个字都有明确的操作含义，不是空洞口号 |
| **层次感** | 同一框架内有空间维度和时间维度的交叉 |
| **助记性** | 设计为可默念、可背诵、可在散步中反复玩味 |
